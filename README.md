# 學習日誌情緒教練 Bot

這是一個 LINE Messaging API Bot 的專案，主題為 **學習日誌情緒教練**。  
使用者每天可以輸入學習心得與心情，系統將進行關鍵字萃取與情緒分析，並提供回饋。  

## 專案目標
- 紀錄使用者每日學習心得與情緒
- 自動進行情緒分析與關鍵字提取
- 提供週期性的學習與情緒回顧

## 環境需求
- Python 3.10+
- Flask（之後使用）
- line-bot-sdk（之後使用）
- python-dotenv（之後使用）
